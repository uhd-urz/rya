# Rya

> [!NOTE]  
> This branch mainly has the fix for typer 0.26. The fix might stop working anytime depending on how many more breaking
> changes Typer makes to its vendored Click. See the [issue #3](https://github.com/uhd-urz/rya/issues/3) to learn more.


Rya is a Python bootstrapper. Rya is made out of the internals of [elAPI](https://github.com/uhd-urz/elAPI). Rya has
been generalized enough so any Python application can be bootstrapped to have the same capabilities as elAPI.

> [!NOTE]  
> Rya is currently in beta.
